Gestión de Biblioteca con Spring Boot, Bootstrap y Thymeleaf

Descripción del Proyecto

Este sistema de Gestión de Biblioteca fue desarrollado como proyecto académico con el objetivo de administrar libros, autores, editoriales y categorías de manera eficiente. La aplicación está construida con Spring Boot, Thymeleaf y Bootstrap 4, ofreciendo una interfaz moderna y fácil de usar.
